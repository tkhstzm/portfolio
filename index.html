coding

出産祝い＆ベビーグッズのセレクトショップのLP

担当範囲

コーディング

使用技術・ツール

HTML / CSS / JavaScript / jQuery
Figma・Visual Studio Code

サイトの紹介

出産祝い＆ベビーグッズのセレクトショップ「cuna select」のリスティング広告で使用するランディングページです。

制作背景

社内のデザイン研修を終え、初めて実務でのコーディングを行ったLPです。社内のデザイナーが作成したデザインカンプを元に1からコーディングを行いました。

苦戦した点

配置が複雑だったため、デザインカンプを忠実に再現するのに苦戦しました。レスポンシブで表示が変わる部分が多かったので、マークアップやクラス名の付け方を工夫し、わかりやすいコードを目指してコーディングを行いました。

感想

初めての実務コーディングで、分からない部分が多く調べながらコーディングを進めたため少し時間がかかりました。実装方法を検索する際の感覚など、1つのLPで学べたことが多く、完成したときは達成感をとても感じました。フロントエンドエンジニアを意識し始めたきっかけになったLPです。
