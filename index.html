about me

takahashi tsuzumi

髙橋 鼓

2000年奈良県生まれ奈良県育ち。
高校卒業後、オーストラリア・ブリスベンに語学留学。 帰国後はWEBサイトの運営・企画に5年間携わっていました。
日々の業務の中でデザインやコーディングに関心がを持ち、社内開発部のデザイン研修に参加。 コーディングの面白さに魅了され、よりコーディングに関われるような職に就きたいと思い退職。
エンジニアとしての知識を身に着けるため、職業訓練校に通い中です。
フロントエンドエンジニアを目指して日々学んでいます。

大学の学園祭にアーティストをブッキングするためのお問い合わせフォーム。

coding

出産祝い＆ベビーグッズのセレクトショップのLP

担当範囲

コーディング（HTML / CSS / JavaScript / jQuery）

使用ツール

Figma・Visual Studio Code

サイトの紹介

出産祝い＆ベビーグッズのセレクトショップのLP

制作背景

リスティング広告で使用するランディングページを制作しました。社内のデザイン研修を終え、初めて実務でのコーディングを行ったLPです。社内のデザイナーが作成したデザインカンプを元に1からコーディングを行いました。

苦戦した点

配置が複雑だったため、デザインカンプを忠実に再現するのに苦戦しました。レスポンシブで表示が変わる部分が多かったので、マークアップやクラス名の付け方を工夫し、わかりやすいコードを目指してコーディングを行いました。

成果

初めての実務コーディングで、分からない部分が多く調べながらコーディングを進めたため少し時間がかかりました。研修では行えなかった柔軟さを　検索する際の感覚など、1つのLPで学べたことが多かったです。フロントエンドエンジニアを意識し始めたきっかけです。
